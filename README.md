# 个人介绍网站

这是一个基于 Vue 3 和 Vite 构建的个人介绍网站，用于展示个人信息、技术能力、项目经历和联系方式。

## 项目内容

- 个人基本信息与求职方向
- 教育背景、语言能力、专业技能和办公技能
- 技术能力展示
- 项目经历展示
- 联系方式与 GitHub 链接

## 技术栈

- 前端框架：Vue 3
- 构建工具：Vite
- 开发语言：JavaScript
- 样式：CSS3
- 部署平台：Vercel
- 代码托管：GitHub

## AI 使用说明

### 1. 使用了哪些 AI 工具

本项目主要使用了 Codex 作为 AI 编程辅助工具，用于协助完成需求拆解、页面结构设计、Vue 组件开发、CSS 样式调整、交互效果实现以及部署流程梳理。

### 2. AI 帮助完成了哪些工作

AI 主要帮助完成了以下工作：

- 生成并调整页面组件结构，包括导航栏、首页展示、关于我、技术能力、项目经历和联系方式等模块
- 协助编写 CSS 样式，实现响应式布局、亮色 / 暗色模式切换、卡片 hover 效果、返回顶部按钮等基础交互

### 3. 自己手动修改了哪些内容

- 页面文案、按钮名称、导航栏名称和联系方式
- 根据页面实际效果多次调整布局、间距、颜色和交互细节
- 网站图标、导航栏头像

### 4. 遇到的问题，以及如何通过 AI 解决

开发过程中遇到的问题主要包括：

- 页面顶部导航栏背景一开始使用模糊效果，滚动到大标题上方时文字被遮挡后显示不自然。通过 AI 分析 CSS 后，将导航背景调整为接近实色的半透明背景，去掉过强的模糊效果。
- 页面底部出现横向滚动条，影响默认整页显示。通过 AI 排查发现是全宽导航背景使用 `100vw` 时在 Windows 浏览器中多计算了滚动条宽度，最终通过限制页面横向溢出解决。
- 导航栏当前区块高亮在滚动到底部和从底部上滑时判断不准确。通过 AI 将原本的 IntersectionObserver 判断改为基于滚动位置的计算逻辑，并增加底部边界处理，使“关于我 / 技术能力 / 项目 / 联系”高亮更加稳定。
